# QR-Code-Menu
Описане.
Мой проект это меню(кафе, бара, ресторана) которое будет отображаться при использовании специального QR кода находящийся на столах заведения.

План.
1. Будут пристувовать блоки разных видов блюд с возможностью добавлять и удалять любое блюдо в списке.
2. Так же возможность добавлять сами блоки со списками блюд.
3. Будет присутвовать возможность регистрации администратора любого заведения.
4. Возможность работы со временем, к примеру: добавлять меню завтрака которое будет появлятся с 8:00 до 11:00 а потом исчезать.
5. Связь с базой данных всех блюд.
